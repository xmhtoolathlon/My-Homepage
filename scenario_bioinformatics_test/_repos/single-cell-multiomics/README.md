# Single-Cell Multi-Omics Analysis

Integrative framework combining scRNA-seq, scATAC-seq, and proteomics data for cancer biomarker discovery.

## Overview
This repository contains the implementation for our paper "Integrative Single-Cell Multi-Omics Analysis for Cancer Biomarker Discovery".

## Features
- Multi-modal data integration (scRNA-seq, scATAC-seq, proteomics)
- Cancer biomarker identification pipeline
- Clinical validation framework
- Visualization tools

## Installation
```bash
pip install -r requirements.txt
```
